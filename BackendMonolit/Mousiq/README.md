# Getting Started

## 1. Build application

```
./build-all.cmd
```

## 2. Run monolith

```
docker-compose up -d
```

**Available at:**
- Backend: http://localhost:8080
- Documentation: http://localhost:8080/swagger-ui.html
- Prometheus: http://localhost:9090
- Grafana: http://localhost:3000


## Stopping

```
docker-compose down
```
